Movieland

(All about movies)

Introduction

Its a website that allows users to see trending movies and search for movies that they are interested in

Click to go to deployed project application

Features

1) Search feature

This feature will allow you to search for movies by keyword

2) Sign-up and Log-in Feature

You will be able to create your own account and log-in anytime

3) adding to favorites feature

You will be able to add your favorite movies to a list of favorites

About the Project

As a movie enjoyer I always dreamed about creating a website that gives me the abiliy to expand my movie knowledge through showing trending movies and the ability to get details for any movie I want , so here I am trying to realise my dream.

This project was created as a portfolio project for Holberton School
